# pytest-automation-framework
This project is a modular automation platform built in Python using Pytest, designed for validating and interacting with Wireless (4G/5G NR) and Satellite Communication systems.The goal is to build a reusable automation core that can integrate with RF instruments, baseband systems, satellite modems, and network nodes.

Unlike traditional test repositories, this framework focuses on:

Clean architecture

Extensible driver layer

Config-driven execution

Scalable test orchestration

Structured logging & reporting
